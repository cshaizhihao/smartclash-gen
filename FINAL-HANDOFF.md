# Clash Smart 分组编辑器 · 最终交付说明

> 交付版本：**V1.0.0**

## 一、项目定位

Clash Smart 分组编辑器是一套面向 **Clash / Mihomo / OpenClash** 的可视化分组与配置生成器。

它聚焦的不是全量代理面板能力，而是以下主链路：

1. 导入机场订阅 / 节点
2. 按中转组 / 落地组进行编排
3. 生成可直接使用的 YAML 与订阅链接

---

## 二、正式推荐安装路径

### 主推荐：Docker 一键部署

推荐原因：
- 不依赖目标机系统 Python 环境
- 避开 `externally-managed-environment` / pip / venv 差异问题
- 更适合作为正式交付与上线部署路径

推荐命令：

```bash
bash -c "$(curl -fsSL https://raw.githubusercontent.com/cshaizhihao/smartclash-gen/main/install-docker.sh)"
```

指定网页端口：

```bash
bash -c "$(curl -fsSL https://raw.githubusercontent.com/cshaizhihao/smartclash-gen/main/install-docker.sh)" -- --web-port 10111
```

### 备用方案：原生安装脚本

适合不使用 Docker 的环境，但稳定性与兼容性不如 Docker 主路径。

```bash
bash -c "$(curl -fsSL https://raw.githubusercontent.com/cshaizhihao/smartclash-gen/main/install.sh)"
```
